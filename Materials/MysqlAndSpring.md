# MySql JDBC in a Spring Project

## Add this to your pom.xml file
````      
<dependency>
	<groupId>mysql</groupId>
	<artifactId>mysql-connector-java</artifactId>
	<scope>runtime</scope>
</dependency>
````    
And click on "Import Changes" in the lower right corner
